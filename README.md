# AMS_EventLink_WebApp

# Install

* Flask
* Flask_sqlalchemy
* Flask_marshmallow
* Marshmallow-Sqlalchemy
* Geopy
```console
$ pip3 install -r requirements.txt 
```  

* Node.js 12.6.1
* npm rebuild node-sass


# Usage

Inside the "react_frontend" folder run:
```console
$ npm run build
```
Inside the "flask_backend" folder run:
```console
$ python3 app.py
```
